# GameKiosk: Board Game Renting System

## Introduction
GameKiosk is an Java GUI application for managing board game rentals. Utilizing Java and JavaFX, it demonstrates technical proficiency and problem-solving skills. The project highlights skills I have demonstrated and learnt through the backend functionality and interactive UI design.

![image](https://user-images.githubusercontent.com/59437991/125469504-d1fde10a-ee4f-46e3-957d-5b13a352b0a4.png)

## Key Features
- **JavaFX GUI**: Responsive, user-friendly interface.
- **MVC Architecture**: Clean, maintainable code.
- **Rental Management**: Add, view, and rent games easily.
- **Stylish Design**: Enhanced with CSS.

## Technologies
- **Java & JavaFX**: Robust backend and dynamic UI.
- **CSS**: Visual appeal.
- **MVC Framework**: Scalable code.

## Quick Start
1. Clone the repo (Optional)
2. Install Java
3. Run `KioskApplication.java`.
